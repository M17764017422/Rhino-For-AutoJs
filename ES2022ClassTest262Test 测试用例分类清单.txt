### 1. 解构赋值 (destructuring-binding) - 1281 个测试

**ES 版本**: ES6
**特性标签**: `destructuring-binding`
**与 Class 关联度**: ❌ 无关（仅借用 class 方法上下文测试解构语法）

**测试目录**:
- `language/statements/class/dstr/` (1920 文件)
- `language/expressions/class/dstr/` (1920 文件)

**测试模式**:
```javascript
// 测试的是解构赋值，不是 class
class C {
  *method([a, b] = [1, 2]) {    // 数组解构
    return a + b;
  }
  method({x, y} = obj) {         // 对象解构
    return x + y;
  }
}
```

**子类别**:
| 子类别 | 测试数量 | 说明 |

```javascript
// 测试 Unicode 转义后的保留字可作为属性名

  cl\u0061ss() { return 2; }     // "class"
}
```

**语法示例**:
```javascript
class C {
  accessor x = 1;           // 实例自动存取器
  static accessor y = 2;    // 静态自动存取器

}
```

**语义等价**:
```javascript
class C {
  #x = 1;
  get x() { return this.#x; }
  set x(value) { this.#x = value; }
}
```

```
**/gen-method/**/*.js     # 生成器方法测试
**/async-method/**/*.js   # 异步方法测试
```

### 2. 如需排除非 ES2022 特性

添加到 `UNSUPPORTED_FEATURES`：
```java
static final Set<String> UNSUPPORTED_FEATURES =
    new HashSet<>(Arrays.asList(
        // ... existing features ...
        "decorators",           // ES2023+
        "top-level-await"       // ES2022 但非 class 特性
    ));
```

### 3. 创建独立的 ES2022 Class 核心测试类

可考虑创建 `ES2022ClassCoreTest262Test.java`，仅包含：
- `elements/**/*.js` (class 元素)
- `static-init-*.js` (静态初始化块)
- `subclass/**/*.js` (子类化)
- `super/**/*.js` (super 关键字)
- `definition/**/*.js` (类定义)
